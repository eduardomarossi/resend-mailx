# resend-mailx

Esse projeto permite imitar o funcionamento do aplicativo Linux para envio de emails `mailx`. Ele aceita os mesmos parametros que o mailx suporta, no entanto, para o envio de emails, ele utiliza o resend-cli atráves do comando resend com os parametros correspondentes.

Essa aplicação que mapeia os commandos e invoca o resend-cli deve ser escrita em Python 3.
